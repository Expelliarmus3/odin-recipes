# ODIN RECIPES 
This repository contains the code for the ODIN RECIPES paper. The code is organized into several directories, each containing different components of the project.
## Directory Structure
- 'recipes': This directory contains the html files for the recipes. Each recipe is stored in a separate html file.They also contain the corresponding images for the recipes.
- 'index.html': This is the main entry point for the project. It contains links to all the recipes and serves as the homepage for the project.
## Usage
To view the recipes, simply open the 'index.html' file in a web browser. From there, you can click on the links to view each individual recipe. Each recipe page will contain the ingredients, instructions, and images for the recipe.
